This document aims to explore the quadratic form for a logitic model:

\[f(X) = logit^{-1}(A + B_1 \cdot X + B_2 \cdot X^2)\] \[f(X) = \frac{1}{1+e^{-(A + B_1 \cdot X + B_2 \cdot X^2)}}\]

# Dt <- data.table(X = seq(from = -5, to = 5, by=0.1))
X <- seq(from=-10, to=10, by=0.1)
A <- c(-20,-10,0,10,20) 
B1 <- c(-10, -5, -1, 0, 1, 5, 10)
B2 <- c(-10, -5, -1, 0, 1, 5, 10)
Dt <- data.table(expand.grid(X, A, B1, B2))
colnames(Dt) <- c("X", "A", "B1", "B2")
Dt$Y <- as.numeric(NA)
Dt[, Y:= 1 / (1 + exp(-(A + B1*X + B2*X^2)))]
# Dt$Y <- 1 / (1 + exp(-(A + B1*Dt$X + B2*Dt$X^2)))

Why these forms?

We take the first derivative of

\[f(X) = \frac{1}{1+e^{-(A + B_1 \cdot X + B_2 \cdot X^2)}}\]

which is

\[f'(X) = \frac{e^{-(A + B_1 \cdot X + B_2 \cdot X^2)} \times (-2B_2 \cdot X - B_1)}{(1+e^{-(A + B_1 \cdot X + B_2 \cdot X^2)})^2}\]

if \(B_2 \neq 0\)

To get \(f'(X) = 0\) we need to have \(X=\frac{-B_1}{2B_2}\).

To know if we have a hump or a dip, we look at the extremum \(Ex\) which is \(A - \frac{B_1^2}{4B_2}\).

The deviation from the intersept is \(\Delta = \frac{-B_1^2}{4B_2}\).

So if \(B_2 > 0\), then \(\Delta \leq 0\) so we have a dip.

And if \(B_2 < 0\), then \(\Delta \geq 0\) so we have a hump.

if \(B_2 = 0\)

\(X=\frac{-B_1}{2B_2}\) is not possible, but we have:

\[f(X) = \frac{1}{1+e^{-(A + B_1 \cdot X)}}\]

which is a sigmoid going up if \(B_1 >0\) and down if \(B_1 <0\).

if \(B_1 = 0\)

The extremum \(\frac{-B_1}{2B_2} = 0\) so the extremum is obtained at 0.
